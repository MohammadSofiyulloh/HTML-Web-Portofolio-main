Front-End Web Developer

Front-End Web Developer adalah Software Developer yang bertanggung jawab untuk merancang dan membangun berbagai aplikasi web secara responsif, interaktif, dan juga user friendly. Seorang Front-End Web Developer memiliki peranan besar dalam pengembangan aplikasi web karena bertanggung jawab langsung kepada pengguna.

Back-End Web Developer

Back-End Web Developer adalah Software Developer yang bertanggung jawab dalam mengelola server, aplikasi, dan database agar dapat saling berkomunikasi dengan baik dan lancar. back end berkewajiban untuk melakukan perhitungan, memproses form, menyimpan data dan juga diharuskan memahami beberapa bahasa pemrograman.
